Instagram Clone

A Mern application where you can add posts, like posts, comment on a post, like a comment, and other CRUD functionalities.
User can also follow/unfollow each other and can interact with their posts

Front end is made with react where state management is done using redux. Bootstrap is used for app design.
Backend is managed using express and node.js and data is stored in Mongodb.
